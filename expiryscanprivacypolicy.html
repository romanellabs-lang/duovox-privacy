# Política de Privacidade — ExpiryScan Última atualização: 26/04/2026 O **ExpiryScan** ("aplicação") respeita a sua privacidade. Esta Política de Privacidade explica que dados podem ser tratados ao utilizar a aplicação. ## 1. Dados recolhidos A aplicação **não recolhe dados pessoais identificáveis** (como nome, e-mail, morada ou contactos) para servidores próprios. A aplicação pode tratar localmente no dispositivo: - imagens da câmara, apenas para leitura de códigos de barras/DataMatrix; - dados descodificados dos códigos (ex.: GTIN, lote, validade), mostrados ao utilizador no ecrã. Esses dados são processados no dispositivo para funcionamento da aplicação. ## 2. Permissões utilizadas ### Câmara (`android.permission.CAMERA`) Usada exclusivamente para: - abrir o leitor de códigos; - capturar imagem para detetar e descodificar códigos de barras/DataMatrix. Sem esta permissão, a funcionalidade de leitura por câmara não funciona. ### Internet (`android.permission.INTERNET`) e estado de rede (`android.permission.ACCESS_NETWORK_STATE`) Podem ser usadas para funcionamento de serviços de terceiros integrados (ex.: anúncios). ## 3. Publicidade (Google AdMob) A aplicação pode apresentar anúncios através do **Google AdMob** (Google LLC). O AdMob pode recolher e tratar dados de acordo com as políticas da Google, incluindo identificadores de dispositivo e dados para medição/desempenho/publicidade. Consulte: - [Política de Privacidade da Google](https://policies.google.com/privacy) - [Como a Google utiliza informações de sites/apps parceiros](https://policies.google.com/technologies/partner-sites) ## 4. Partilha de dados O desenvolvedor não vende dados pessoais. Eventual tratamento por terceiros (ex.: AdMob) segue as respetivas políticas. ## 5. Retenção de dados A aplicação não mantém base de dados remota própria com dados pessoais do utilizador. Dados lidos podem permanecer apenas no contexto de utilização local da aplicação. ## 6. Crianças A aplicação não é direcionada especificamente a crianças. Se acreditar que houve tratamento inadequado de dados de menores, contacte-nos para avaliação. ## 7. Segurança São adotadas medidas razoáveis para reduzir riscos, mas nenhum método de transmissão/armazenamento é 100% seguro. ## 8. Alterações desta política Esta política pode ser atualizada periodicamente. A data de "Última atualização" será revista quando houver alterações. ## 9.
